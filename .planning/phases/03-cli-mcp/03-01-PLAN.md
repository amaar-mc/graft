<task type="auto" tdd="true">
  <name>Task 2: Create shared indexing pipeline</name>
  <files>src/indexer/pipeline.ts, tests/indexer/pipeline.test.ts</files>
  <behavior>
    - buildIndex(rootDir) calls discoverFiles -> checks cache -> parseFiles if stale -> buildGraph -> computePageRank -> writeCache -> returns { graph, scores, files, results }
    - buildIndex with personalization param passes it through to computePageRank
    - Cache-hit path: readCache -> isCacheValid -> deserializeResults -> buildGraph -> computePageRank (skip parseFiles + writeCache)
    - Cache-miss path: full pipeline with writeCache at the end
    - Returns readonly types matching FileGraph, PageRankResult shapes
  </behavior>
  <action>
Create `src/indexer/pipeline.ts` exporting:

```typescript
interface IndexResult {
  readonly graph: FileGraph;
  readonly scores: ReadonlyMap<string, number>;
  readonly files: readonly string[];
  readonly results: readonly ParseResult[];
}

async function buildIndex(
  rootDir: string,
  personalization?: ReadonlyMap<string, number>,
): Promise<IndexResult>
```

Implementation flow:
1. `const files = await discoverFiles(rootDir)`
2. `const cached = await readCache(rootDir)`
3. If cached is non-null and `await isCacheValid(cached, rootDir, files)` is true:
   - `const results = deserializeResults(cached, rootDir)`
   - Skip to step 5
4. Else: `const results = await parseFiles(files)` then `await writeCache(rootDir, results, files)`
5. `const graph = buildGraph(results)`
6. `const { scores } = computePageRank(graph, { alpha: 0.85, maxIterations: 100, tolerance: 1e-6, personalization })`
7. Return `{ graph, scores, files, results }`

PageRank options use the same defaults as Phase 2 tests (alpha 0.85, maxIterations 100, tolerance 1e-6).

Write tests in `tests/indexer/pipeline.test.ts` that mock discoverFiles, parseFiles, readCache, writeCache, isCacheValid, deserializeResults to verify:
- Cache-hit skips parseFiles and writeCache
- Cache-miss calls parseFiles and writeCache
- Personalization is passed through to computePageRank
- Return shape has all four fields

Use vi.mock() to mock the modules. Do NOT perform real file I/O in these tests.
  </action>

  </verify>
