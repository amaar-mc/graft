<objective>
Implement the filesystem cache that serializes parsed results to `.graft/cache.json` with mtime+size fingerprinting for invalidation.

Purpose: On a 100K LOC repo, re-parsing every file on every run is wasteful. The cache stores ParseResult[] and fingerprints so subsequent runs skip parsing when files haven't changed. This is the performance foundation — fast restart is essential for the `npx graft` zero-friction experience.

Output: `src/cache/index.ts` with read/write/validate functions, plus tests.
</objective>

  <action>
    1. Create `src/cache/index.ts`:
       - `CACHE_VERSION = 1` constant (bump to invalidate all caches on schema change)
       - `CACHE_DIR = '.graft'` and `CACHE_FILENAME = 'cache.json'`

       - `interface FileFingerprint { readonly mtimeMs: number; readonly size: number; }`

       - `interface CacheEntry { readonly version: number; readonly rootDir: string; readonly createdAt: string; readonly fingerprints: Record<string, FileFingerprint>; readonly parseResults: readonly SerializedParseResult[]; }`
         - `SerializedParseResult` is identical to `ParseResult` but with relative filePaths (and nodes' filePaths also relative). Define a local type for this.

       - `async function computeFingerprint(filePath: string): Promise<FileFingerprint>`:
         - `const stat = await fs.stat(filePath); return { mtimeMs: stat.mtimeMs, size: stat.size };`

       - `async function writeCache(rootDir: string, results: readonly ParseResult[], filePaths: readonly string[]): Promise<void>`:
         - Create `.graft/` directory with `mkdir({ recursive: true })`
         - Compute fingerprint for each filePath
         - Convert absolute paths in results to relative (relative to rootDir) using `path.relative()`
         - Build CacheEntry object with version, rootDir basename, createdAt ISO string, fingerprints (keyed by relative path), serialized results
         - Write JSON with `fs.writeFile(cachePath, JSON.stringify(entry, null, 2))`

       - `async function readCache(rootDir: string): Promise<CacheEntry | null>`:
         - Read `.graft/cache.json` — return null if ENOENT
         - JSON.parse — return null if malformed (wrap in try/catch)
         - Basic shape validation: check `version` field exists
         - Return parsed CacheEntry

       - `async function isCacheValid(cached: CacheEntry, rootDir: string, currentFiles: readonly string[]): Promise<boolean>`:
         - Return false if `cached.version !== CACHE_VERSION`
         - Convert currentFiles to relative paths
         - Return false if file count differs from fingerprint count
         - For each current file: compute fingerprint, compare mtime+size against cached fingerprint. Return false on any mismatch.
         - Return false if stat throws (file deleted)
         - Return true only if ALL match

       - `function deserializeResults(cached: CacheEntry, rootDir: string): readonly ParseResult[]`:
         - Convert relative paths back to absolute using `path.resolve(rootDir, relativePath)`
         - Return ParseResult[] with absolute paths restored

    2. Create `tests/cache/cache.test.ts`:

       - Create real temp files with known content, stat them
       - Test all behaviors above
       - Clean up temp dirs in afterEach
       - For mtime change tests: write to a file, wait 10ms, write again to change mtime
