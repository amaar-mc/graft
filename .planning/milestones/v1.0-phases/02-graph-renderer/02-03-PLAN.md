    - Test: Files appear sorted by PageRank score descending (highest first)
    - Test: Each file shows its relative path and score
    - Test: Definitions (functions, classes, etc.) appear indented under their file, excluding import/export kinds
    - Test: Output ends with token count footer in format "[~N tokens]"
    - Test: Empty graph produces only the token footer

    Budget enforcement:
    - Test: Output with budget=2048 does not exceed 2048*3=6144 chars (excluding footer)
    - Test: Files that would push output over budget are excluded entirely (no partial files)
    - Test: With a very small budget (e.g., 100 tokens), only the highest-ranked file appears
    - Test: With a very large budget, all files appear

    JSON renderer:
    - Test: Output is valid JSON (JSON.parse succeeds)
    - Test: JSON contains files array with filePath, score, definitions
    - Test: JSON contains metadata object with fileCount, edgeCount, tokenCount
    - Test: JSON files array is sorted by score descending
    - Test: JSON definitions exclude import/export kinds
  </behavior>
  <action>
    1. Create `src/renderer/tree.ts` with `renderTree(graph: FileGraph, scores: ReadonlyMap<string, number>, rootDir: string, options: TreeRendererOptions): string`:
       - Compute charBudget = tokenBudget * charsPerToken
       - Sort files by score descending
       - For each file (in rank order):
         a. Compute relative path via `path.relative(rootDir, filePath)`
         b. Build header line: `{relativePath} [score: {score.toFixed(4)}]`
         c. Build definition lines: filter out import/export kinds, format as `  {kind} {name} (L{startLine})`
         d. Join header + defs + blank line into block
         e. If totalChars + block.length > charBudget: break (do NOT include partial blocks)
         f. Append block, accumulate totalChars
       - Compute tokenCount = Math.ceil(totalChars / charsPerToken)
       - Append footer: `[~{tokenCount} tokens]`

    2. Create `src/renderer/json.ts` with `renderJson(graph: FileGraph, scores: ReadonlyMap<string, number>, rootDir: string, options: JsonRendererOptions): string`:
       - Build structured object:
         ```
         {
           metadata: { fileCount, edgeCount, tokenCount, rootDir },
           files: [{ filePath (relative), score, definitions: [{ name, kind, startLine, endLine }] }]
         }
         ```
       - Files sorted by score descending
       - Definitions exclude import/export kinds
       - edgeCount = sum of all forwardEdges set sizes
       - tokenCount = Math.ceil(JSON.stringify(output).length / 3)
       - Return JSON.stringify(output, null, 2)

    3. Create test files:
       - `tests/renderer/tree.test.ts`: rendering correctness, score ordering, definition display
       - `tests/renderer/budget.test.ts`: budget enforcement, boundary cases
       - `tests/renderer/json.test.ts`: JSON structure and validity
